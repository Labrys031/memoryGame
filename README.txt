Black Cat Memory Game

A simple match game with an animated cat companion. Note: Cat not availible on smaller displays.

How to play

Select any card. Card will reveal image. Attempt to find match by selecting other cards. If correct, match will stay flipped. If incorrect both cards will return to starting position. Moves are counted once two cards have been flipped. Timer starts after first move. Complete all matches in 11 moves or less to get three stars.
